# БАГАТОВИМІРНА БЕЗУМОВНА ОПТИМІЗАЦІЯ
## Методи оптимізації та дослідження операцій

> [!NOTE]
> Основним призначенням програми є вирішення задач багатовимірної безумовної оптимізації.
>
> Програма призначена для знаходження екстремумів цільових функцій багатьох змінних без обмежень на область визначення.
>
> Метою створення цього додатку є надання користувачам зручного інструменту для автоматизації процесу пошуку оптимальних розв’язків складних математичних задач та демонстрація практичного застосування методів чисельної оптимізації.

### Appumo - UMO Application

Клас Appumo створює інтерфейс користувача для комфортного налаштування параметрів багатовимірної безумовної оптимізації й візуального відображення результатів розв’язку задач.

### UMO - Unconstrained Multivariable Optimization

Головною задачею класу UMO є розв’язання задач багатовимірної безумовної оптимізації.

Перелік методів багатовимірної безумовної оптимізації, які реалізовані:
- Хука-Дживса
- Нелдера-Міда
- найшвидшого спуску
- спряжених градієнтів
- Квазі-Ньютона (BFGS)
- Ньютона
